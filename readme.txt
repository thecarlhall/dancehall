OpenSocial service implementations for BuddyPress.

== Description ==

DanceHall add RESTful interfaces that can be used with the provided OpenSocial services
implementations to turn a BuddyPress installation into an OpenSocial information provider.

== Installation ==

This plugin requires WP-RESTful be installed. Please see
http://wordpress.org/extend/plugins/wp-restful/ for details in installing WP-RESTful.

Once you have WP-RESTful installed, install this plugin by

1. Extract into your plugins directory.
2. Enable the plugin in the WP admin.

You should now have "WP-RESTful" at the bottom of the administration menu. Click on "WP-RESTful"
-> "Server" to verify that you have each of the expected controllers: Activites, Albums,
Application Data, Groups, Media Items, Messages and People.
